# PA Restaurant Inspection Analysis (Shiny App)

This project is an interactive R Shiny application built to explore Pennsylvania restaurant inspection data.

The app allows users to:
- Filter inspections by location and risk level
- Explore inspection outcomes and trends
- Interact with visualizations without writing code
